Welcome to the RWD Retrofit demo site

Its purpose is to demo how you can successfully retrofit an existing site (including ones built using old-skool images for everything) to have a modern, responsive version for smaller screens.

When used in conjunction with our RWD Retrofit plugin, you can force the original “desktop” site on to large tablets (should you wish).

View the unminified versions of the CSS and JavaScript to see how it's accomplished and read the article on Smashing Mag for more info.
